# VR Sound Visualizer

## Installation

1. Install Node.js
2. Clone this repository

   ```bash
   git clone https://github.com/8gaU8/vr-sound-visualizer
   ```

   or open this repository with VS Code.

3. Install dependencies

   ```bash
   cd vr-sound-visualizer
   npm install
   ```

4. Run vite server

   ```bash
   npm run dev
   ```

5. Open your browser

   URL will be shown in the terminal.
